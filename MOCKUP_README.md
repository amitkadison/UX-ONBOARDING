# 🎨 Onboarding Mockup - DUMMY DATA בלבד!

פרויקט פשוט לעיצוב ה-Onboarding Flow ללא API, רק עם DUMMY DATA.

## 🚀 הרצה מהירה

```bash
npm install
npm run dev
```

פתח: http://localhost:3000

## 📁 מה יש כאן?

- ✅ כל הקומפוננטות מהפרויקט המקורי ב-`components/onboarding/`
- ✅ UI Components מ-shadcn/ui
- ✅ Tailwind CSS מוגדר
- ✅ TypeScript
- ✅ RTL Support

## 🎨 מה לעשות עכשיו?

1. הרץ `npm run dev`
2. עצב את הדפים ב-`app/page.tsx`
3. שנה צבעים ב-`app/globals.css`
4. התאם קומפוננטות ב-`components/onboarding/`
5. התאם UI ב-`components/ui/`

## 📝 הערות

- כל הנתונים הם DUMMY DATA
- אין צורך ב-API endpoints
- פשוט עצב ותראה שינויים מיידית!

## 💡 טיפים

  /* ...וכו' */

```bash

```
